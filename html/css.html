¿Que es CSS

CSS en español significa "Hojas de estilo en cascada" y es un lenguaje de diseño grafico para definir y crear la presentacion de un documento estructurado escritgo en lenguaje marcado

¿Que podemos hacer con CSS?

Podemos definir el color,fuente,alineación del texto, tamaño, bordes, espacio, capas y muchas otras caracteristicas tipograficas

¿Como comenzo?

Se creó como una solución para separar el contenido de la presentación en la web. Antes de su creación, HTML se utilizaba tanto para estructurar el contenido como para definir su estilo visual.Sin embargo, este enfoque resultaba limitado y poco eficiente, especialmente a medida que las páginas web se volvían más complejas y variadas en diseño

¿Quien lo creó

Fue propuesto por Hakon Wium Lie y Bert Bos en 1994 como una forma de permitir a los desarrolladores web seprar el diseño visual del contenido estructural de una pagina web. Esto permitió una mayor flexibilidad y control sobre el aspecto del mismo, facilitando la creación de diseños mas complejos y consistentes

¿Es "CSS" importante?

CSS se ha convertido en un estándar fundamental en el desarrollo web moderno, permitiendo una mejor accesibilidad, mantenibilidad y rendimiento de los sitios web.
